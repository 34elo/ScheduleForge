**ScheduleForge** - ваша автоматизированная система составления расписаний

Бот представляет собой инструмент для администраторов кофейни и её сотрудников. Администраторы могут связываться с
сотрудниками, составлять графики автоматически и редактировать их в ручную. Сотрудники могут записываться на смены и
связываться с администраторами.

Доска проекта - https://www.figma.com/board/wX2rVUEchtCchamo6suJX5/Black-Fox?node-id=0-1&t=fYujuNjgdX6otlul-1

P.S Название гпт придумал. Типо составитель расписаний
